# NUnit spike

A simple proof of concept for test automation using NUnit and C#. The project was developed using Microsoft .NET SDK 8.0 in Debian 12 (Bookworm).

The first version of this spike was created following the guide:
[learn.microsoft.com/en-us/dotnet/core/testing/unit-testing-csharp-with-nunit](https://learn.microsoft.com/en-us/dotnet/core/testing/unit-testing-csharp-with-nunit).

From a command line:

```
dotnet test
```
